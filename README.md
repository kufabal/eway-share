# EWAY SHARE

이화인을 위한 안심 택시 쉐어링 앱

## 기능

- 이화여대 인증 기반 로그인/회원가입
- 신분 선택 (학부생, 대학원생, 교직원, 교수)
- 신분 노출 설정
- 목적지 기반 매칭 시스템
- 실시간 매칭 상태 표시

## 시작하기

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

브라우저에서 `http://localhost:5173`을 열어 확인하세요.

### 빌드

```bash
npm run build
```

## 기술 스택

- React 18
- Vite
- Tailwind CSS
- Lucide React (아이콘)
